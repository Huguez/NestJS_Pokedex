#  Run in development

1.- Clone the repository
```
git clone https://github.com/Huguez/NestJS_Pokedex.git
```
2.- Install dependencies
```
npm install
``` 

3.- Set up the database with Docker
```
docker-compose up -d
```

4.- Run the seed
```
http://localhost:3000/api/v2/seed
```

5.- Run the service in development
```
npm run dev
```
