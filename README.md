# Ejecutar en desarrollo

1. Clonar el repositorio
```
git clone ...
```
2. Ejecutar:
```
npm install
``` 
3. Tener Nest Cli instalado
```
npm i -g @nest/cli
```

4. Levantar la base de datos con docker
```
docker-compose up -d
```

5. ejecutar el comando
```
npm run dev
```
